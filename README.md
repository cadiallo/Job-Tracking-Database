# Job Tracking Database

## Overview

Designed and developed a SQL Server database system to manage customers, workers, vendors, materials, skills, and job records. The database uses normalized relational tables, primary keys, foreign keys, and SQL queries to ensure data integrity and efficient data management.

## Features

- Customer management
- Worker management
- Vendor management
- Material tracking
- Skill tracking
- Job management
- Relational database design
- Foreign key relationships
- Data integrity enforcement

## Technologies Used

- Microsoft SQL Server
- SQL
- SQL Server Management Studio (SSMS)

## Skills Demonstrated

- Database Design
- SQL Development
- Relational Database Modeling
- Normalization
- Primary and Foreign Keys
- Query Development
- Data Integrity Management

## Database Entities

- Customers
- Workers
- Vendors
- Materials
- Skills
- Jobs
- Worker Skills
- Job Materials
